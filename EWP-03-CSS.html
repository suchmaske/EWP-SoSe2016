Klassen & IDs

Klassen

Gleiche Eigenschaften für mehrere, bestimmte Elemente

Beispielklasse: Schwarzes Auto

Schwarzer BMW vs. Schwarzer Audi

  • Lack: Schwarz
  • Reifen: 4
  • Türen: 4
  • Getriebe: Manuell

Beispiel in HTML: Buttons

Verwendung von Klassen

  • class-Attribut im Element
  • Mehrere Klassen im selben Attribut durch Leerstelle getrennt
Das Button-Element hat die Klassen btn und btn-success.
  • Selektion über: Punkt + Klassenname: .classname
.btn-success {
  background-color: green;
  color: white;
}CSS für HTML-Elemente mit der Klasse: btn-success

IDs

Merkmale für besondere, einzelne Elemente

Eine ID sollte nur 1 Mal vorkommen!

Beispiel: Roter BMW Sportwagen

Fügt weitere Attribute zur "Schwarzes Auto" Klasse hinzu oder ändert sie

  • Türen: 2 vier
  • Farbe: rot schwarz
  • Reifen: 4
  • Getriebe: manuell

Beispiel in HTML: Buttons

Verwendung von IDs

  • id-Attribut im Element
  • Nur eine ID pro Element
  • ID sollte im Code nicht zwei Mal vorkommen!
Das Button-Element hat die ID self-destruct.
  • Selektion über: Raute + Klassenname: #idname
#self-destruct {
  background: black;
  font-size: 30px;
}CSS für das HTML-Elemente mit der ID: self-destruct

Vergleich: Klassen & IDs

Klassen ID
Attribut in HTML class id
Selektorzeichen in CSS . #
Verwendungshäufigkeit Beliebig oft 1 Mal pro Seite

  • Selektoren mit mehreren Bedingungen
  • Child-Elements

Verschiedene Selektoren: ,

Selektoren durch Komma trennen

.btn-success, .btn-error  {
  opacity: 0.3;
}Elemente mit den Klassen .btn-success oder .btn-error sind stark durchsichtigt.

Mehrere Bedingungen

Selektor soll mehrere Klassen oder IDs haben.

Selektoren ohne Leerzeichen hintereinanderweg schreiben

.btn.btn-success {
  font-size: 50px;
}Nur Elemente mit sowohl der Klasse .btn als auch der Klasse .btn-success
haben eine Schriftgröße von 50px.

 a > .highlight {
  background-color: yellow;
