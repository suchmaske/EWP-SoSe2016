Warmup

HTML

Hypertext Markup Language

  • CERN 1992: Technologie, um schnell Forschungsergebnisse miteinander zu teilen
  • Dokumente sind miteinander verbunden
  • Hyperlinks zum Springen zu anderen Hypertextdokumenten

Mehr als nur Text

  • Lineare Struktur wird aufgebrochen
  • Verlinkungen zu anderen Medien
  • Andere Medien (Bild, Video, Ton) einbinden
  • Riesiges Informationsnetzwerk

Warum nicht XML?

  • XML: eXtensible Markup Language
  • SGML > XML > HTML
  • HTML ist eine spezifische Auslegung von SGML bzw. XML für die Darstellung von Hypertext-Dokumenten

Struktur, Darstellung, Funktion

Typ Technologie
Struktur HTML HyperText Markup Language
Darstellung CSS Cascading Stylesheets
Funktion JS JavaScript
